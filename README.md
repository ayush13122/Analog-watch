# ⏱️ Analog Watch

A simple and elegant Analog Watch built using HTML, CSS, and JavaScript. This project displays a real-time working clock with hour, minute, and second hands just like a traditional analog watch.

## 🚀 Features

- Real-time clock functionality
- Smooth movement of hour, minute, and second hands
- Clean and responsive UI design
-  Lightweight and fast
-  Works on all screen sizes (Responsive)

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript

## 📂 Project Structure
Analog-watch/
│── index.html
│── style.css
│── script.js
│── assets/

💡 How It Works

The analog clock is created using JavaScript by calculating rotation angles for each hand based on the current time:

Hour hand → rotates every hour
Minute hand → rotates every minute
Second hand → rotates every second

This is typically done using trigonometric/time-based calculations.
